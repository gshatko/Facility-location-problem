# Facility-location-problem

Репозиторий посвящён  оптимизации инфраструктуры и логистических решений последней мили в городской среде. В основе работы лежит последовательное изучение теоретических подходов к оценке современных форматов городской логистики, а также разработка проектного решения на основе методов пространственного моделирования и задачи размещения объектов. В прикладной части обоснована и продемонстрирована концепция цифрового сервиса выбора локаций для объектов городской логистики и микромобильности на основе геоданных OpenStreetMap, ETL-процедур, векторизованных расчетов расстояний и алгоритмов оптимизации. Показано, что сочетание методов геоинформационного анализа, Facility Location Problem и многоцелевой оптимизации позволяет формировать решения, ориентированные одновременно на покрытие спроса, транспортную доступность и возможность практического внедрения в городской среде
